# Importadora Web

E-commerce de accesorios importados con inventario propio, pagos en linea, entrega local/nacional, panel administrador, SEO tecnico y arquitectura preparada para futura app movil.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- MySQL/MariaDB
- Mercado Pago
- Google Login

## Desarrollo local

```bash
npm install
npm run dev
